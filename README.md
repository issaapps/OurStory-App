# 💜 OurStory-App

A private couple mobile application designed to capture, preserve, and relive meaningful moments in a secure and interactive environment.

---

## 📱 Overview

**OurStory-App** is a personal mobile app that creates a private digital space for couples to connect, communicate, and store their most valuable memories.

It blends emotional experience with modern mobile development to deliver a secure and beautifully designed platform.

---

## ✨ Features

* ⏳ Love Counter (years, months, days, seconds)
* 💬 Real-time Private Chat
* 📸 Memories Gallery (organized with dates & categories)
* 💌 Daily Love Messages
* 👤 Identity-based UI experience
* ☁️ Firebase Real-time Sync
* 🔐 Biometric Authentication

---

## 🛠️ Tech Stack

* **Java (Android Development)**
* **Firebase (Authentication, Realtime Database, Storage)**
* **Material Design UI**
* **Glide (Image Loading)**
* **AndroidX Biometric**

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/issaapps/OurStory-App.git
```

2. Open the project in **Android Studio**

3. Connect Firebase:

* Add your `google-services.json`
* Enable Authentication & Database

4. Run the app 🚀
